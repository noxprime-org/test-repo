# test-repo

A minimal Node.js starter project scaffold, set up for testing.

## Prerequisites

- [Node.js](https://nodejs.org/) 18 or newer (uses ES modules and the built-in test runner — no external dependencies required)

## Usage

Run the example entry point:

```bash
npm start
```

Run the tests:

```bash
npm test
```

## Project structure

```
.
├── src/
│   └── index.js     # Entry point with example functions
├── test/
│   └── index.test.js # Tests using node:test
├── .editorconfig
├── .gitignore
├── LICENSE
├── package.json
└── README.md
```
